# Map component

Renders Yandex.Maps API [Map][] object

```jsx
// @live

const mapState = {
  center: [55.751574, 37.573856],
  zoom: 9,
};

const MapObject = () => (
  <YMaps>
    <Map defaultState={mapState} />
  </YMaps>
);

render(<MapObject />);
```

> `WARN` Yandex.Maps Map parent element should have at least some size set to
> it, otherwise the Map instance is rendered into the container with zero size.
>
> To avoid that, this library uses `width` and `height` props as default way of
> sizing the map element, but then if the library user provides `style` or
> `className` prop, Map component will assume that the Map instance is sized by
> those props and will not use `width` and `height` props
