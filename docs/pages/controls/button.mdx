# Button

Renders Yandex.Maps API [Button][] control

```jsx
// @live

const mapState = {
  center: [55.751574, 37.573856],
  zoom: 3
};

const ButtonExample = () => (
  <YMaps>
    <Map defaultState={mapState}>
      <Button 
        data={{ content: 'Button' }}
        onClick={() => console.log('Click!')}
      />
    </Map>
  </YMaps>
)

render(<ButtonExample />);
```
