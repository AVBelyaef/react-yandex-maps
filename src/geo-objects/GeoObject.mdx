# Map

Renders Yandex.Maps API [GeoObject][docs] object

## Basic Usage

<Playground>
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 3,
      }}
    >
      <GeoObject
        geometry={{
          type: 'LineString',
          coordinates: [[55.76, 37.64], [52.51, 13.38]],
        }}
        options={{
          geodesic: true,
          strokeWidth: 5,
          strokeColor: '#F008',
        }}
      />
    </Map>
  </YMaps>
</Playground>
