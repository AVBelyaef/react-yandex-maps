# Panorama component

Renders Yandex.Maps API [Panorama][] object

```jsx
// @live

const PanoramaObject = () => (
  <YMaps>
    <Panorama coordinates={[55.733685, 37.588264]} width='100%' height='100%' />
  </YMaps>
);

render(<PanoramaObject />);
```

> `WARN` Yandex.Maps Panorama parent element should have at least some size set to
> it, otherwise the Panorama instance is rendered into the container with zero size.
>
> To avoid that, this library uses `width` and `height` props as default way of
> sizing the map element, but then if the library user provides `style` or
> `className` prop, Panorama component will assume that the Map instance is sized by
> those props and will not use `width` and `height` props
