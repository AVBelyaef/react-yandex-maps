# RoutePanel

Renders Yandex.Maps API [RoutePanel][control] control

```jsx
// @live

const mapState = {
  center: [55.751574, 37.573856],
  zoom: 9,
};

const RoutePanelExample = () => (
  <YMaps>
    <Map defaultState={mapState} height={320}>
      <RoutePanel
        options={{
          showHeader: true,
          title: 'Build a route!',
        }}
      />
    </Map>
  </YMaps>
);

render(<RoutePanelExample />);
```
